这是一个简单的基于Boost/asio开发的，在控制台上实现的聊天室项目，希望对你有所帮助。

学习Boost/asio就是跟着官方网站的tutorial来看的，只看那个对于写一个基于控制台的简单的聊天室程序就够用了。

学习线程也是掌握一下相关的概念就OK，甚至我其实根本没用着std::thread相关的东西，boost库都很好地进行了封装，直接使用就OK。

聊天室程序是基于client和server模型来开发的，client将消息转发给server，然后由server来将信息转发给另一个client。

第一个版本是clientA定时向server发送信息，然后server直接把信息回传给clientA。

第二个版本引入了异步的键盘输入，就可以顺利的通过键盘输入来发送消息了。

第三个版本其实和第二个版本是一个东西，只是我去掉了帮助调试的输出消息。第二个版本就能够完成端到端的对话了。
